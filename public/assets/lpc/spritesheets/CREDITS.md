# LPC Character Asset Credits

DeskRPG's character customization and sprite composition system uses assets from the
Liberated Pixel Cup / Universal LPC Spritesheet Character Generator project.

- Upstream repository:
  https://github.com/LiberatedPixelCup/Universal-LPC-Spritesheet-Character-Generator
- Upstream licensing and attribution guide:
  https://github.com/LiberatedPixelCup/Universal-LPC-Spritesheet-Character-Generator#licensing-and-attribution-credits
- Full upstream credits file bundled with this project:
  [CREDITS.csv](./CREDITS.csv)

Important notes:

- The images under this directory are not covered by DeskRPG's project license alone.
- LPC sprite assets are distributed under mixed licenses, including CC0, CC-BY,
  CC-BY-SA, OGA-BY, and GPL, depending on the individual source asset.
- If you redistribute DeskRPG with these character assets, keep the bundled
  `CREDITS.csv` file accessible to users and preserve upstream attribution.
- If you extract, modify, or redistribute only a subset of the character assets,
  you are still responsible for complying with the licenses that apply to those
  specific files.
